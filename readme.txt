== Description ==
Gemfind Diamond Tool is a WordPress plugin that allows you to list and filter diamonds based on third-party APIs. It seamlessly integrates with WooCommerce, making it easy to display diamonds on your website.

== Installation ==

1. Upload the 'gemfind-diamond-tool' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure that WooCommerce plugin is installed and activated. If not, you will be prompted to activate it.
4. Configure the plugin settings through the 'Gemfind Diamond Tool' menu in the WordPress dashboard.
5. Use the [gemfind-diamond-tool] shortcode in your pages to display the diamond search functionality.

== Features ==

- List and filter diamonds based on various parameters.
- Seamlessly integrate with WooCommerce.
- Display diamonds on your website using shortcodes.
- Customizable settings to configure diamond listing and display.

== Plugin Settings ==

The plugin can be configured through the 'Gemfind Diamond Tool' menu in the WordPress dashboard. You can set the dealer ID, API URLs, and other options related to diamond listing and display.

== Shortcode Usage ==

To display the diamond search functionality, use the following shortcode in your pages or posts:

[gemfind-diamond-tool]

== Frequently Asked Questions ==

1. Can I customize the appearance of the diamond listing?
Yes, you can apply custom CSS to modify the appearance of the diamond listing.

2. Can I use this plugin without WooCommerce?
No, this plugin requires WooCommerce to function correctly.
